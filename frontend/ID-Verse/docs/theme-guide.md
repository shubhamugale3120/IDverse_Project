# IDverse Theme Guide

## Colors
- Navy Blue: #0a192f (background)
- White: #ffffff (content/cards)
- Accent Cyan: #64ffda (buttons/links)

## Typography
- Primary: Inter (modern, clean)
- Alternative: Poppins (rounded, readable)

## Layout Style
- Card-based design
- Rounded corners (8px / 16px)
- Soft shadows
